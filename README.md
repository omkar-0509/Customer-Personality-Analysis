# Customer Personality Analysis

This project performs **Customer Personality Analysis** using real-world marketing data.  
The goal is to understand **customer behavior, spending patterns, and campaign performance** using industry-standard data analysis tools.

## Business Objective
- Understand customer demographics
- Identify high-value customers
- Analyze product spending behavior
- Evaluate marketing campaign performance
- Improve business decision-making

## Dataset
- 2240 Customers
- 29 Features
- Data Includes:
  - Customer Profile
  - Income Information 
  - Product Spending
  - Purchase Behavior
  - Campaign Response
